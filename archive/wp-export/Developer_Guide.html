Source Code

Code for all Sana developed components of the platform are available on the Sana Github site. Please see individual component pages in the documentation pages for full details on  obtaining and building the components from source.

Contributing

Sana is an open source platform and community contributions are highly encouraged and appreciated. If you are interested in contributing, please obtain a GIthub account, fork the source code of the specific component you would like to contribute to, and submit pull requests with any changes. Please see the Contributing page for additional information.
